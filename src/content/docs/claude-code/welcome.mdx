This is the Advanced Track. Claude Code is the same Claude you already know, working directly in the files and folders on your computer instead of a browser chat. If you have not used the claude.ai Core Track yet, that is the place to start. This track assumes you are already comfortable with Claude and ready for hands-on work.

You will not be asked to write code or learn anything technical. You will be asked to let Claude work where your files actually live, and to approve each change before it happens.

## Who this is for

You are a Private Equity professional who is comfortable with claude.ai and now wants Claude to work directly in your own files and folders: tidying data packs, checking models, turning a folder of memos into clean summaries. That is exactly what this track is for.

You do not need to be technical, and you do not need to write code. Everything here is plain-English instructions and a simple yes or no from you.

## What this track covers

The track moves in a deliberate order, one module at a time:

- **Getting Started**: what Claude Code is, and how it compares to claude.ai.
- **Install and Setup**: getting the free VS Code editor onto your machine and signing in.
- **First Sessions**: your first real run, and the approval loop where you review every change.
- **Good Habits**: the CLAUDE.md rulebook and the habits that keep you safe.
- **Saving and Shipping**: saving your work with Git in plain English, and going live when you are ready.
- **PE Recipes**: concrete use-cases you can lift straight into your week.

## How to work through it

Go top to bottom the first time. Each page builds on the one before it, so the setup and the safety habits are in place before you do any real work. Once you are set up, you can jump straight to the recipe you need and come back for the rest later.

<Callout type="confidential" title="You stay in control">
Your files stay on your machine. Claude Code proposes every change and waits for you to approve it, one at a time. Nothing goes live until you choose to publish it.
</Callout>

## Where to go next

When you are ready, start with **What Claude Code Is**, the first page of Getting Started. It explains what Claude Code is in plain terms before you install anything.
