# KPMG × Claude for Private Equity

A polished teaching wiki that takes KPMG Private Equity professionals from Claude.ai basics to confident mid-level use — covering context, Projects, PE playbooks, and connectors.

Built by **NXT Partners AI** for **KPMG**.

## What this is
An educational web wiki (not the API, not Claude Code) focused on the **Claude.ai web app** for PE professionals using **Claude Enterprise/Team**. Content runs from foundations through practical, copy-paste PE playbooks.

## Tech stack
- **Astro** — content-first static site (minimal JS, fully custom design)
- **GSAP** — motion (hero, scroll reveals, micro-interactions)
- **Pagefind** — static search
- **Markdown/MDX** — content authoring
- Deploy target: GitHub Pages (portable to KPMG internal hosting)

## Develop
```bash
npm install
npm run dev
```
